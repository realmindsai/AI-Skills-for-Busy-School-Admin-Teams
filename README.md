# AI Skills for Busy School Admin Teams

Workshop materials for practical AI training designed for school administration staff.

**Workshop date:** 5 March 2026
**Presenters:** Tracy Anthony & Dr Dennis Wollersheim, [Real Minds AI](https://realmindsai.com.au)

## Workshop Site

**[View the workshop materials page](https://realmindsai.github.io/AI-Skills-for-Busy-School-Admin-Teams/)**

## What's Included

### Introductory Track (Exercises 1-8)
- Slide deck (HTML, PDF, PPTX, Marp source)
- Participant prompt pack
- Marraburra Secondary College data pack (fictional)

### Advanced Track (Exercises A1-A5)
- Advanced slide deck (HTML, PDF, PPTX, Marp source)
- Advanced prompt pack
- Messy mini-pack (intentionally broken data for verification exercises)

## About

This workshop covers practical AI tool usage for school admin teams: email drafting, document summarisation, spreadsheet help, data verification, privacy and governance considerations, and compliance checking.

All school data is **entirely fictional**. Marraburra Secondary College does not exist.

Slides are built with [Marp](https://marp.app/) — the Markdown source files are included.
